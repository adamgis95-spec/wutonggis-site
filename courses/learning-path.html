学习路径 ​

阶段 1：基础与环境 ​

  • 认识坐标系、投影、数据类型
  • 完成长距离数据清洗与质量检查
  • 安装 QGIS/ArcGIS/必备 CLI 工具

阶段 2：分析与制图 ​

  • 练习缓冲区、叠加分析、网络分析
  • 制作出版级地图与Web地图
  • 建立项目模板与样式规范

阶段 3：开发与部署 ​

  • 学习 Leaflet / Mapbox GL / deck.gl 基础
  • 构建数据管线：获取、处理、发布
  • 自动化部署与性能监测

阶段 4：项目落地 ​

  • 选择一个真实场景完成端到端案例
  • 输出沉淀文档与复盘
  • 规划进阶方向（时空大数据、3D、机理模型）
