# 工具与插件

## 必备工具
- QGIS：开源桌面 GIS，全流程分析制图。
- GDAL/OGR：批处理投影转换、切片、格式互转。
- Tippecanoe：将 GeoJSON 转为矢量切片，适合 Web 可视化。
- Mapbox Studio / Maputnik：在线样式设计。

## 浏览器端插件
- Leaflet 控件：比例尺、图层控制、测距。
- Mapbox GL 插件：Draw、Geocoder、Inspect、Terrain。
- Deck.gl Layers：点云、热力、弧线、网格聚合等高阶可视化。

## 使用建议
- 优先选择活跃维护项目
- 查看许可证是否满足商业/开源要求
- 做好缓存与日志，便于问题追踪
