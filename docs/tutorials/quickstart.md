# 快速开始

## 目标
在本节中，你将完成环境准备并渲染第一幅在线地图。

## 前置条件
- 安装 Node.js 16+ 与 npm/pnpm/yarn
- 浏览器支持 WebGL
- 获取一份示例矢量数据（GeoJSON）

## 步骤
1. 克隆示例项目：`git clone https://github.com/example/wutonggis-starter.git`
2. 安装依赖：`npm install`
3. 启动开发服务：`npm run dev`
4. 打开浏览器访问提示的本地地址，确认地图正常显示。

## 检查点
- 地图底图可以平移缩放
- 数据图层加载成功且符号化正确
- 控件（比例尺、缩放、图例）显示正常

## 下一步
继续阅读[可视化与制图](/tutorials/visualization)优化地图表现。
