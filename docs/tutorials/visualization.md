# 可视化与制图

## 制图思路
- 先确定地图用途与读者，再选择视觉变量（颜色、形状、大小）。
- 优先简化：减少视觉噪音，突出核心图层。

## 配色与符号
```css
:root {
  --wutong-primary: #0f766e;
  --wutong-accent: #f97316;
}
```
- 使用主色强调交互要素，辅色用于高亮警示或关键区域。
- 连续数值建议采用单色渐变，分类数据使用对比色系。

## 布局与注记
- 保持视觉层次：标题 > 主体图层 > 辅助元素。
- 注记避免遮挡关键要素，可使用描边或遮罩提升可读性。
- 导出前检查比例尺、指北针、数据来源与时间戳。

## 工具示例
- QGIS/ArcGIS：符号化、版式设计、批量导出。
- Mapbox GL/Leaflet：Web端互动制图。
- SLD/Mapbox Style：建立可复用的样式规范。
